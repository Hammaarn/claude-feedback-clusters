# Behavioral

Add rules here about how the user wants to be communicated with, their cognitive style, what they value in collaboration, and interpersonal patterns.

Example topics this cluster handles:
- How verbose responses should be
- Whether the user wants pushback or compliance
- How to handle uncertainty (hedge openly vs. commit and revise)
- What kinds of suggestions are welcome vs. unwanted
- The user's working style and attention preferences
